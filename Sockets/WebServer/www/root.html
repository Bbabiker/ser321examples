You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /commits -- returns all commits messages for a given repository, parameters are 'rp=OWNER/REPO ' and 'br=BRANCH NAME EXAMPLE: /comm?q=bbabiker/assign1git&s=newbranch--Will print all comits' messages for NEWBRANCH
  • /encode -- returns a simple encoded string, parameters are 's1=SOME_STRING ' and 'n=ENCODING_CHARACTER EXAMPLE:/encode?s1=babiker&n=2--Will print 'b2a2b2i2k2e2r2
