You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply? -- multiply two numbers, 'num1=FIRST_OPERAND' and 'num2=SECOND_OPERAND'
    EXAMPLE: /multiply?num1=5 & num2=5 --Will return 25
  • /github? -- returns all public repositories for a given user, parameter 'query=users/OWNER/repos ' users/amehlhase316/repos EXAMPLE: /github?query=users/amehlhase316/repos--Will print all public repositories for amehlhase316
  • /commits? -- returns all commits messages for a given repository, parameters are 'rp=OWNER/REPO ' and 'br=BRANCH NAME EXAMPLE: /comm?q=bbabiker/assign1git&s=newbranch--Will print all comits' messages for NEWBRANCH
  • /encode? -- returns a simple encoded string, parameters are 's1=SOME_STRING ' and 'n=ENCODING_CHARACTER EXAMPLE:/encode?s1=babiker&n=2--Will print 'b2a2b2i2k2e2r2
