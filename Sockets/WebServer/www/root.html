You can make the following GET requests

  • /file/sample.html -- returns the content of the file sample.html
  • /json -- returns a json of the /random request
  • /random -- returns index.html
  • /multiply?num1=5&num2=5 --prints '25'
  • /github?query=users/amehlhase316/repos --prints all public repos for 'amehlhase316'
  • /commits?rp=bbabiker/assign1git&br=newbranch --prints all commits messages for NEWBRANCH
  • /encode?s1=babiker&n=2 --prints 'b2a2b2i2k2e2r2'
